# 易算 —— 东方传统文化智能分析平台

融合传统玄学智慧与现代技术，提供八字命理、紫微斗数、姓名学、奇门遁甲、风水堪舆、择日等功能的一站式命理分析平台。

## 功能模块

- **八字命理**：专业的四柱八字排盘与大运流年分析，15个分析维度
- **紫微斗数**：十二宫位排盘，四化星曜，三方四正分析
- **姓名学**：五格剖象、三才配置、康熙笔画分析
- **奇门遁甲**：九宫八门排盘，局势分析，行动建议
- **风水堪舆**：罗盘、八宅、飞星、阳宅阴宅
- **择日系统**：黄历宜忌、建除十二神、二十八宿
- **知识库**：16篇深度教程，覆盖八字/紫微/奇门/风水/择日

## 技术栈

- 前端：Next.js 14 + React + TypeScript + Tailwind CSS
- 后端：NestJS + Prisma + SQLite
- 图表：Recharts
- 动画：Framer Motion

## 本地运行

```bash
# 安装依赖
pnpm install

# 启动后端
cd backend && pnpm dev

# 启动前端
cd frontend && pnpm dev
```

打开 http://localhost:3000 即可使用。

管理员后台：/admin，密码可自行配置。
